## 宝石链简介
![mahua](http://www.baoshichain.org/images/logo.png)
* 宝石链将用区块链技术，打造一个革命性的游戏生态系统。BSC将成为游戏虚拟世界的比特币。
* 整个宝石链生态系统将围绕：用户虚拟资产交易所、游戏跨境交易，游戏支付，电子竞技，玩家社区等内容展开
* 游戏用户：为用户打通游戏虚拟资产导出通道，并保障虚拟资产交易流转无风险。同时通过区块链技术，将游戏用户变成游戏收益的享有者，玩家在娱乐的同时获得更多的价值回报。
* 游戏厂商：通过共享收益获得更多铁杆游戏玩家，减少渠道投入成本， 将更多资金用到游戏开发及回馈玩家，实现一个健康循环的游戏社区。
